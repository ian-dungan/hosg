# Trees Folder

Place tree models here for all biomes.

**Kenney - Nature Kit**

## Files to Add

Rename tree models from download packs to:
- **Oak_01.glb** - Oak tree (grassland, forest)
- **Oak_02.glb** - Oak tree variant
- **Pine_01.glb** - Pine/conifer (snow biomes)
- **Willow_01.glb** - Willow (swamp biomes)

## After Upload

URLs will be:
```
https://raw.githubusercontent.com/ian-dungan/hosg/main/assets/environment/trees/Oak_01.glb
https://raw.githubusercontent.com/ian-dungan/hosg/main/assets/environment/trees/Pine_01.glb
```

Already configured in `hosg_asset_loader.js`!
